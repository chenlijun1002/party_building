#### 项目架构
项目架构说明

src/router 路由所在

src/store 存储

src/service 所有接口请求方法

src/pages 所有页面及其私有组件

src/common 公用组件

src/config 公用方法配置 baseUrl等

src/assets 静态资源 css js
